# Wendy Lead Workflow

**Owner:** Wendy Delgado — both Yucca Valley and Desert Hot Springs  
**Hours for callbacks:** Mon–Thu 8:00 AM–4:30 PM · Fri 9:00 AM–2:00 PM · Sat/Sun Closed  

## Where leads come from

1. **Website forms** → email to office Hotmail (YV or DHS) + thank-you page for the patient  
2. **Hotmail inboxes** you monitor (`hartdentalyv@hotmail.com` / `hartdental02@hotmail.com`)  
3. **Phone calls** to each office  
4. **Walk-ins**

## Your daily tool (no Microsoft license required)

1. Open Google Sheets (free) **or** Excel on the office computer.  
2. File → Import → upload `ops/leads/lead-workbook.template.csv` from this project (or use the shared copy Lindsay/Ryan provides).  
3. Add **one row per lead** the same day you see it.  
4. Update **Status** and **Next follow-up** every time you touch the lead.

Statuses allowed: `New` · `Contact Attempted` · `Contacted` · `Appointment Scheduled` · `Follow-Up Required` · `No Answer` · `Won` · `Lost` · `Spam/Test`

## When a website email arrives

1. Open it. If the name/message says **TEST LEAD — DO NOT CONTACT**, mark **Spam/Test** and stop.  
2. Otherwise create/update a workbook row: location, service, phone, email, source=`website`.  
3. Call (or text if they asked for text) within your SLA (implants/emergency first).  
4. Log **Last contact** and set **Next follow-up**.  
5. When booked, set status **Appointment Scheduled** and fill appointment date.

## Duplicate check (30 seconds)

Before adding a row, search the sheet for the same phone or email from the last 30 days. If found, update that row instead of creating a new one — put the new inquiry in **Notes**.

## Privacy

- Do not paste chart notes, diagnoses, or insurance ID numbers into the sheet or marketing email.  
- Keep the sheet access limited to Wendy + Lindsay + Dr. Hart (and Ryan for tech).  
- Export a backup CSV weekly to a secure office folder (not personal email).
