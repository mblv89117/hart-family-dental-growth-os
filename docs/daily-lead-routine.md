# Daily Lead Routine (Wendy)

**Before patients / first 30 minutes**

1. Open YV Hotmail + DHS Hotmail.  
2. Open lead workbook.  
3. Process overnight/weekend web leads (Sat/Sun closed — handle Monday morning).  
4. Flag **Emergency** and **Implant** rows first.

**Throughout the day**

5. Log every phone inquiry and walk-in in the workbook (source = phone / walk-in).  
6. Update statuses after each attempt.  
7. Check **Next follow-up** filter for overdue rows — call those before new cold leads when possible.

**End of day (10 minutes)**

8. Filter status = `New` or `Follow-Up Required` — none should be blank on next follow-up date.  
9. Count today’s New → Contacted → Appointment Scheduled (write numbers in a daily note for Lindsay if asked).  
10. Export/backup sheet if it’s Friday.

**Response-time goal (business hours)**

| Type | Target first touch |
| --- | --- |
| Emergency | Immediate / next open slot call |
| Implants | < 15 minutes when possible |
| Straightening / new patient | < 30 minutes when possible |
| After hours / weekend | Next business morning |
